1. システム概要
   本システムは、Webカメラを用いた顔認証により、従業員の出勤・退勤を記録・管理するクラウド型アプリケーションです。

URL: https://attendance-system-52qp.onrender.com/
推奨環境: Google Chrome, Safari, Edge (PC/スマホ対応)

※LINEなどから直接開くことはできません、ブラウザから使用してください



2. 【管理者向け】初期設定・管理画面の使い方
   各企業の担当者（管理者）が行う操作です。

2-1. ログイン
システムURLへアクセスすると、ログイン画面が表示されます。
発行された ログインID と パスワード を入力し、「ログイン」ボタンを押します。
2-2. メインメニュー（ダッシュボード）
ログインに成功すると、打刻画面（カメラ画面）が表示されます。 画面右上の 「⚙️ 管理」 ボタンを押すと、管理ダッシュボードへ移動できます。

2-3. 管理ダッシュボードの機能
機能	説明
🏢 会社設定	システムのヘッダーに表示される会社名を変更できます。
📊 勤怠データ出力	全期間の勤怠履歴をCSVファイルでダウンロードします。給与計算ソフト等で利用できます。
👥 従業員管理	登録されている従業員の一覧を表示します。「削除」ボタンで退職者などのデータを消去できます。
🔒 パスワード変更	管理者自身のログインパスワードを変更します。セキュリティのため、初回ログイン後に必ず変更してください。
2-4. ログアウト
画面右上の「ログアウト」ボタンを押すと、システムからログアウトし、ログイン画面に戻ります。





3. 【従業員向け】出勤・退勤の使い方
   日々の打刻操作です。

3-1. ユーザー登録（初回のみ）
画面上部の入力欄に 自分の名前 を入力します。
カメラに顔を向け、緑色の枠が表示されるのを確認します。
「登録」 ボタンを押します。
※「登録完了」と表示されればOKです。
3-2. 出勤・退勤
カメラに顔を向けます。
画面左上に 「🟢 OK: \[あなたの名前]」 と表示されたら認識成功です。
「出勤」 または 「退勤」 ボタンを押します。
音声アナウンスと共に「完了」のポップアップが出れば記録されています。
💡 音声操作: マイクを許可している場合、「出勤します」「退勤します」と話しかけるだけでも打刻できます。





4. 【開発者向け】システム保守・管理ツール (Super Admin)
   運営者（開発者）が、データベースを直接操作して「企業の追加・削除」や「状況確認」を行うためのCLIツールです。

4-1. 準備
ローカル環境（PC）のターミナルで以下を実行し、依存ライブラリを準備します。

npm install pg bcrypt
4-2. 起動方法
ターミナルで以下のコマンドを実行します。

node super\_admin.js
4-3. メニュー詳細
1 会社一覧を表示
現在登録されている全ての会社を表示します。id （会社ID）の確認に使用します。

2 新しい会社を追加
新規クライアントと契約した際に使用します。

会社名（例: 株式会社ABC）
管理者ログインID（例: admin\_abc）
管理者パスワード（例: pass1234）
✅ 完了後: クライアントに上記IDとパスワードを伝えれば、即座に利用開始できます。

3 会社の社員を確認
特定の会社に紐づく従業員リストを表示します。データ確認用です。

4 会社を削除 (⚠️危険)
契約終了時などに、会社とその関連データ（管理者、社員、勤怠履歴）を全て物理削除します。

確認: 最後に yes と入力しない限り実行されません。
注意: 一度削除すると復元できません。

5 管理者パスワードをリセット クライアントがパスワードを忘れた場合に使用します。

入力: 対象の会社ID、新しいパスワード

完了後: 新しいパスワードをクライアントに通知してください。

6 管理者ID(ユーザー名)を変更 クライアントからログインIDの変更希望があった場合に使用します。

入力: 対象の会社ID、新しいログインID
